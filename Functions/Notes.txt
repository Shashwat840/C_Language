Elements of UDF 

    There are three elements in UDF 
    1) Funtion Declaration(Name of function)
    2) Funtion calling 
    3) Funtion Defination

1) The function defination is an independent prog that is written as per requirement

2) The function calling(Invoke) is required when we need to resuse that function by it's 
name 

3) The declaration of function is required for declaration of variables
