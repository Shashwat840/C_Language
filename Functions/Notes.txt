Elements of UDF 

    There are three elements in UDF 
    1) Funtion Declaration(Name of function)
    2) Funtion calling 
    3) Funtion Defination

1) The function defination is an independent prog that is written as per requirement

2) The function calling(Invoke) is required when we need to resuse that function by it's 
name 

3) The declaration of function is required for declaration of variables 

Function Declaration(Function Prototype)

Passing parametres to the function 
    Diffrence between call by value and call by refrence 
    
    Call by value :- 
    1) The "copy" of actual parametre are passed into formal Parametre
    2) Change in formal parametre will not result in change in actual parametre
    3) Pointers are not used

    Call by refrence(&):- 
    1) "Refrence" to actual parametre are passed into formal parametres
    2) Change in formal parametre will result in change in actual parametres
    3) Pointers are used
