Passing parametres to the function 
    Diffrence between call by value and call by refrence 
    
    Call by value :- 
    1) The "copy" of actual parametre are passed into formal Parametre
    2) Change in formal parametre will not result in change in actual parametre
    3) Pointers are not used

    Call by refrence(&):- 
    1) "Refrence" to actual parametre are passed into formal parametres
    2) Change in formal parametre will result in change in actual parametres
    3) Pointers are used

Scope of variables
    Local variables:- 
    1) Declared inside the function body 
    2) Not initilised automatically
    3) use of local varible are advicible 

    Global variables:- 
    1) Declared outside the function body
    2) initilised automatically by value 0 or dependent on type
    3) Too much use of Global variables makes the program diffcult

The pre Processors
    It is a program that processes the source code before it passes throught the 
    compiler 
    
    It operates under the control of what it is know as pre processer command lines
    they all begins with symbol # and do not require a semicolon at the end 

    This directives can be divided into three parts 
        1) MicroSubstitution directives
        2) File inclusion directives    
        3) Complier controlled directives

    1) MicroSubstitution means to replace the micro by the microstrings 
        ex--> #define macro_name string; 
              #define max 10;
        Here the pre processer relaces the max everywhere in the prog by 10 
        micro is mainely used for fast execution

    2) File inclusion starts from #include 
        #include <filename>
            ex-> #include<stdio.h>
    
    3) Complier control in ANSI more pre processers are used for conditions 
        ex #if , #elif , #else

What is recursion?
    It is a process in which a "fun calls itself" directly or indirectrly to solve a 
    problem 

    It divides a problem into smaller chunks of the same type  
    
    For this process we required two cases:- 
        1) Recursive case :- Fun calls itself 
        2) Base case:- Some termination condition 
    
    Because if the process is not terminating at some point then it will go into infinite loop 
-------------------------------------------------------------------------------------------
Advantages and disadvantages of recursion:- 
    1) Complex can be easily retured in less code 
    2) By using recursion infinite set of objects can be defined in finite way 

disadvantages of recursion:- 
    1) It uses more memory so genrally it is slow 
    2) Temination condition is lost thus it will go in infinite loop
