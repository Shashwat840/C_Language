Diffrence between dynamic memory and static memory

Static memory:- 
    1) Performed at compile time 
    2) Assigned to stack 
    3) Size must be known at compile time 
    4) It is best if the required size of memory known in advance 

Dynamic memory:- 
    1) Performed at run time
    2) Assigned to heap  
    3) Size may be unknown at compile time 
    4) It is best if we don't have idea about how much memory is required
